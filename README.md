# water_sort_puzzle
ウォーターソートパズル解答プログラム

[「ウォーターソートパズルの答えが知りたい」](https://happyclam.github.io/software/2025-04-27/waterbottle)

~~~Ruby
# ウォーターボトルをBoxに配置
box = Box.new(
  [
    Bottle.new([ORANGE, RED, ORANGE, RED]),
    Bottle.new([BLUE, BLUE, ORANGE, RED]),
    Bottle.new([RED, ORANGE, BLUE, BLUE]),
    Bottle.new,
    Bottle.new
  ]
)
box.display
~~~
